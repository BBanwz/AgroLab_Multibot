Книга по взаимодеюствию с обучающим набором AgroLab Multibot с модулем Poluslab
==================================================================================

В представленной книге описано использование обучающего набора. Для правильного и последовательного понимания использования набора книга разделена на несколько глав:

* В первой главе вы узнаете, из чего состоит конструкция AgroLab Multibot.
* Во второй главе вы изучите что такое кинематика набора.
* В третьей главе вы вы познакомитесь с интерфейсом управления набора.
* В четвертой главе вы изучите написание программного кода для платы нижнего уровня.
* В пятой главе вы вы изучите все доступные модули.
* В шестой главе вы обучитесь основам использования дополнения PolusLab.
